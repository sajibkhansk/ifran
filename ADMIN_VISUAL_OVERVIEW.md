## Quick Start

1. **Start the admin server:**
   ```bash
   npm run admin
   ```

2. **Test it works:**
   ```bash
   curl http://localhost:5002/api/admin/health
   ```

3. **Import Postman collection:**
   - Open Postman
   - Import `postman_collection.json`
   - Start testing!

4. **Read the docs:**
   - `ADMIN_BACKEND_SETUP.md` - Overview
   - `ADMIN_API_DOCUMENTATION.md` - Full docs
   - `ADMIN_QUICK_REFERENCE.md` - Quick commands
